Recommended 

git clone
git@github.com:jadjoubran/laravel5-angular-material-starter.git
cd laravel5-angular-material-starter
cp .env.example .env #create .env file from .env.example. Fix DB credentials & APP_KEY
npm install -g gulp bower
composer update
php artisan jwt:generate
npm install
bower install
gulp && gulp watch
php -S localhost:8081 -t public
You're ready to go! Open http://localhost:8081 in your browser.
